# Aplicação desktop barcode

Instale o ghostscript no seu computador: https://www.ghostscript.com/download/gsdnld.html

<p>Tipo de barcode:</p>

    * qrcode: QR Code;
    * azteccode: Aztec Code;
    * pdf417: PDF417;
    * interleaved2of5: Interleaved 2 of 5;
    * code128: Code 128;
    * code39: Code 39;
    * ean13: Ean 13.
